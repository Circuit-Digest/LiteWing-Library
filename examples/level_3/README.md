# Level 3 — Advanced Control

Deep-dive into the drone's control systems. These examples build on
Level 2 flight skills.

| # | Script | What it teaches |
|---|--------|-----------------|
| 01 | `01_position_hold_analysis.py` | Analysis Position Hold Perfomance and Log the data |
| 02 | `02_movement_commands.py` | forward/backward/left/right with speed control |
| 03 | `03_waypoint_navigation.py` | `fly_to()` and `fly_path()` for absolute position nav |
| 04 | `04_keyboard_control.py` | WASD keyboard control with hold modes |
| 05 | `05_pattern_navigation.py` | Fly geometric shapes using the built-in shape methods |

## Prerequisites

- Complete Level 2 examples first
- Comfortable with arm → takeoff → land sequence
- Know where `emergency_stop()` is!
